MathArts
在做什么？

MathArts 围绕中国数术建设开放源码生态。目前公开项目包括核心基础设施、天文历法与紫微斗数领域引擎、可复用 Agent Skills，以及用于组织协作的治理文档和自动化工具。

组织政策与项目规则，
分别维护；
同一问题只保留一个权威来源。

职责明确 · 公开优先 · 证据优先

做了哪些事情？

MathArts 的公开工作分为领域引擎、生态基础设施与组织治理。每个项目都可以直接进入 GitHub 阅读代码、文档与讨论。

组织如何协作？

MathArts 的组织治理明确区分组织政策与项目规则，并以公开记录、证据和清晰责任边界支持协作。

  1. 01

    公开优先

    不涉及安全、隐私或人员保护的决定，在可追溯的 Issue、Discussion 或 Pull Request 中作出。

  2. 02

    证据优先

    技术和领域判断尽可能引用定义、资料、测试、推导或可复现结果。

  3. 03

    最小权限

    组织、仓库、自动化和发布权限，只授予完成职责所需的主体。

  4. 04

    职责明确

    组织政策与项目规则分别维护，不建立含混的中间权威层。
